# 🗺️ 旅游规划助手

一个功能强大的在线旅游规划工具，基于Google Maps API，支持智能路线规划、多平台导航集成和完整的行程管理功能。

![旅游规划助手](https://img.shields.io/badge/版本-v2.0-blue) ![Google Maps](https://img.shields.io/badge/Google%20Maps-API-green) ![响应式](https://img.shields.io/badge/响应式-✓-success)

## ✨ 核心特性

### 🎯 智能规划系统
- **🔍 智能搜索**：支持地点名称、类型搜索，实时获取详细信息
- **📍 一键添加**：点击地图直接添加自定义地点
- **🎯 游玩/待定**：灵活的地点状态管理，支持游玩列表和待定列表
- **📝 自定义编辑**：为每个地点自定义名称和添加备注信息
- **🔄 拖拽排序**：直观的拖拽操作调整游玩顺序

### 🚗 专业导航集成
- **🟢 高德地图**：国内导航首选，精准路况信息
- **🔵 谷歌地图**：全球通用，海外旅行必备
- **📊 真实距离**：使用Google Distance Matrix API计算实际驾驶距离和时间
- **🛣️ 路线可视化**：地图显示完整路线，带方向箭头和清晰编号
- **⚡ 智能优化**：使用贪心算法自动优化游玩路线

### 🏙️ 智能管理功能
- **🏙️ 城市过滤**：自动识别城市，支持单城市/多城市切换显示
- **📋 便捷操作**：一键复制地点名称、地址，支持个人导航
- **💾 方案管理**：保存/加载/删除多个旅游方案，支持方案对比
- **📤 导入导出**：支持数据备份和HTML分享版导出
- **🎨 视觉优化**：现代化界面设计，隐藏滚动条，优化用户体验

## 🚀 快速开始

### 方式一：直接使用（演示模式）
1. 下载项目文件到本地
2. 双击打开 `index.html` 文件
3. 立即开始使用基础功能（搜索、列表管理等）

### 方式二：完整功能配置

#### 🔑 API密钥配置（推荐）
获得完整功能需要配置Google Maps API密钥：

1. **获取API密钥**
   - 访问 [Google Cloud Console](https://console.cloud.google.com/)
   - 创建项目并启用API服务：
     - Maps JavaScript API
     - Places API
     - Geocoding API
     - Static Maps API
   - 创建API密钥并设置使用限制

2. **应用配置**
   - 打开应用，点击右上角 **⚙️ 设置**
   - 在"API密钥配置"中输入您的Google API密钥
   - 点击"保存设置"
   - **🎉 配置完成！刷新页面后API将自动应用**

#### 💡 智能配置特性
- ✅ **自动应用**：配置一次，每次打开自动加载
- ✅ **安全存储**：API密钥仅存储在您的浏览器本地
- ✅ **智能提示**：未配置时显示友好的配置引导
- ✅ **动态加载**：配置后立即生效，无需手动刷新

## 🎯 功能指南

### 📍 搜索与添加地点
```
搜索地点 → 查看详情 → 添加到游玩列表
      ↓
点击地图 → 自动获取地址 → 添加到列表
```

### 🎮 游玩列表管理
- **📝 编辑地点**：点击✏️编辑名称和备注
- **🎯 状态切换**：在"游玩"和"待定"状态间切换
- **📍 快速定位**：点击📍在地图上居中显示
- **📋 便捷复制**：复制地点名称或完整地址
- **🧭 个人导航**：从当前位置导航到指定地点
- **🗑️ 删除管理**：移除不需要的地点

### 🚗 路线规划与导航
- **距离计算**：自动计算真实驾驶距离和时间
- **路线优化**：智能重排地点顺序，最小化总距离
- **导航集成**：每段路线支持选择高德或谷歌地图导航
- **可视化路线**：地图显示完整路径，带方向指示

### 🏙️ 城市管理
- **自动识别**：从地址自动提取城市信息
- **过滤显示**：支持查看全部城市或单个城市的地点
- **地图联动**：过滤时地图自动聚焦到选中城市

### 💾 数据管理
- **本地存储**：自动保存所有数据，刷新不丢失
- **方案管理**：保存多个旅游方案，便于对比选择
- **数据备份**：导出JSON格式备份文件
- **分享功能**：导出包含地图截图的HTML文件

## 🚨 重要提醒

### API使用
- 📊 **免费额度**：Google Maps API提供一定免费使用量
- 💳 **计费设置**：超出免费额度需要启用计费，建议设置用量限制
- 🔒 **安全配置**：建议为API密钥设置域名和使用量限制
- 🌐 **网络要求**：需要稳定的网络连接访问Google服务

### 数据安全
- 🔐 **本地存储**：所有数据仅存储在您的浏览器中
- 🚫 **无服务器**：不会向任何外部服务器上传您的旅游数据
- 💾 **数据备份**：建议定期导出数据进行备份

## 🆘 常见问题

<details>
<summary><strong>Q: 页面显示"API配置提示"怎么办？</strong></summary>

A: 这表示尚未配置Google Maps API密钥：
1. 点击提示中的"立即配置"按钮
2. 或点击右上角⚙️设置按钮
3. 输入您的API密钥并保存
4. 配置一次后，每次打开都会自动应用
</details>

<details>
<summary><strong>Q: 地图无法显示或搜索功能不工作？</strong></summary>

A: 请检查以下几点：
- API密钥是否正确配置
- 是否启用了所需的API服务（Maps JavaScript API、Places API等）
- 网络连接是否正常
- 浏览器是否支持Google服务
</details>

<details>
<summary><strong>Q: 为什么需要启用计费？</strong></summary>

A: Google Maps API从2018年开始要求验证付款方式：
- 提供一定额度的免费使用量
- 正常使用通常不会产生费用
- 建议设置每日配额限制以控制成本
</details>

<details>
<summary><strong>Q: 距离计算不准确？</strong></summary>

A: 本应用使用Google Distance Matrix API计算真实驾驶距离：
- 基于当前交通状况
- 考虑实际道路情况
- 实际行程可能因路况变化有所差异
</details>

<details>
<summary><strong>Q: 移动端使用体验如何？</strong></summary>

A: 已针对移动端进行全面优化：
- 响应式设计适配各种屏幕
- 触控操作优化
- 建议使用Chrome或Safari浏览器
</details>
